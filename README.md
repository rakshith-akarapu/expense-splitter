# Expense Splitter

A React-based web application that helps groups split expenses easily and calculate the final settlement between members.

## 🚀 Live Demo
https://easy-split-vert.vercel.app

## Features
- Add group members
- Add shared expenses
- Automatic equal expense calculation
- Final settlement calculation
- LocalStorage support
- Reset all data
- Responsive UI

## Tech Stack
- React (Vite)
- Tailwind CSS
- JavaScript
- LocalStorage

## Run Locally

Clone the project

```bash
git clone https://github.com/rakshith-akarapu/expense-splitter.git
```

Install dependencies

```bash
npm install
```

Start the project

```bash
npm run dev
```
